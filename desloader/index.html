Desloader

What is it?

DesLoader is a plugin for Desmos that allows you to save graphs locally, import OBJ models into Desmos3D, load arrays from CSV files, and code in Desmos. DesLoader is mainly built for Desmos3D but some parts can be used in normal Desmos. Documentation can be found here.

Here is some of the stuff it can do:

  • Upload 3D models into Desmos3D with textures
  • Create graphs using a custom programming language called descode that is like JS
